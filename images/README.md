# images/

Reserved for locally hosted image assets.

All images currently used across the site are hotlinked from the original
organization's WordPress media library (moabacademiademusica.com). To make this
site fully independent, download those assets into this folder and update the
`<img>` / `<figure>` `src` attributes in `index.html` and the files under `pages/`
to point here instead (e.g. `images/team/cesar-garcia.jpg`).
